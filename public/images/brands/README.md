## Image Requirements:
- Format: PNG (transparent background preferred)
- Size: 200x200 pixels minimum
- Quality: High resolution, clear logos
- Background: Transparent or white

## How to Add Logos:
1. Save each logo file with the exact filename above
2. Place in: `public/images/brands/`
3. The app will automatically use them in the sidebar
